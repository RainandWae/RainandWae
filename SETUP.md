# Setup

1. Create a public GitHub repository named exactly `RainandWae`.
2. Upload or push every file from this folder into that repository.
3. Open the repository's Actions tab and run `Starry Night Contributions` manually once.
4. After the workflow finishes, your GitHub profile will show the animated SVG from the `output` branch.

GitHub profile READMEs only render when the repository name matches the account name exactly. For your account, the required repository is:

```text
RainandWae/RainandWae
```

This folder is already initialized as a local Git repository with the first commit made. After creating the empty GitHub repo, push it from this machine with:

```powershell
cd C:\Users\HP\Downloads\DockerPractice\github-profile-starry-night
git remote add origin https://github.com/RainandWae/RainandWae.git
git push -u origin main
```
